- [ ] P7-T8 后端：管理员种子脚本（创建初始管理员）
- [ ] P7-T9 前端：`admin/` 工程初始化（Vite + Vue 3 + TS）
- [ ] P7-T10 前端：Vant 组件库接入 + 移动端 viewport 配置
- [ ] P7-T11 前端：Pinia + 路由 + 请求封装（与 wx-app 相同思路，独立 Token key）
