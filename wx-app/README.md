# pet-visit 小程序前端

uni-app + Vue 3 + TypeScript + Pinia。

## 启动开发环境

1. 安装依赖：

```bash
pnpm install
```

2. 启动 H5 调试：

```bash
pnpm dev:h5
```

或微信小程序开发者工具调试：

```bash
pnpm dev:mp-weixin
```

## 环境变量

`.env.development` / `.env.production` 中配置 `VITE_API_BASE_URL`。
